Detailed Description

template<typename T>
struct framework::serializable::serializable_default_specification< T, typename std::enable_if< is_default_value_serializable< T >::value, void >::type >

Default value_type specification.

Serializes the field identified by

get_value_name <T>::type

using the specification given by

get_value_specification <T>::type

The specification is assumed to identify the field's underlying value type, as specified by

typename type_extractor <typename get_value_specification <T>::type>::type
Note
The methods provided here access the implementation's method directly though the reference provided by interface.
