Detailed Description

template<typename T, typename Enabler = void>
struct framework::serializable::serializable_default_specification< T, Enabler >

Default container serializer.

Provides default base container type serializers. The default base container serializers tests must be mutually exclusive - new base container types should wrap this test in a unique is_default_NAME_serializable trait. In addition, an is_default_serializable override must be provided to disable the missing specification/fundamental type specializations provided here.

Example: container_type enables it's specialization of this template only if

is_default_container_serializable <T>::value == true

and extends is_default_serializable so that

!is_default_container_serializable <T>::value || is_default_serializable <T>::value
Note
No default is provided here; a missing serializable_default_specification announced by is_default_serializable will trigger an undefined template error. The implementation below is provided for documentation purposes only.
