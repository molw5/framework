Detailed Description

template<typename T>
struct framework::serializable::big_endian< T >

Serializes the child type T using big endian byte ordering. FRAMEWORK_HOST_ENDIANNESS and FRAMEWORK_HOST_FLOAT_ENDIANNESS define the host endianness; see framework/common/endian.hpp.

Note
T is required to be a fundamental type as defined by std::is_fundamental
